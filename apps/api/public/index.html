Solution

The settlement layer for AI-cited knowledge.

ProofSource sits between content owners and AI applications so knowledge can be licensed, cited, verified, paid for, and audited without one-off negotiations.

What breaks today

AI uses knowledge, but the commercial path is missing.

Creators and publishersLose traffic, attribution, and monetisation when AI systems use their work as answer fuel.
AI product teamsNeed trusted, licensed, citable sources without manually negotiating with every content owner.
EnterprisesNeed evidence trails: what source was used, whether rights cleared, what was paid, and who approved it.

Developers

Add paid, verified sources to an AI workflow.

Developers use ProofSource through an SDK or REST API inside a RAG pipeline, agent workflow, AI search product, or enterprise research assistant.

How developers use it

The app asks, ProofSource clears, the model cites, settlement follows.

01

Create a project

Register the AI app, set team budgets, approved source categories, maximum price per citation, and webhook endpoints.

02

Search licensed sources

At retrieval time, ask ProofSource for relevant sources with price, licence, reputation, and policy metadata.

03

Authorize source use

Before the model sees premium content, request authorization. ProofSource checks budget, licence terms, and usage policy.

04

Verify and settle

After generation, submit the answer and citation spans. ProofSource verifies real use, settles the payment, and returns a receipt.

SDK example

Designed to fit where RAG already happens.

A developer should not rebuild licensing, wallet routing, citation verification, and receipts. ProofSource becomes the payment-aware source layer around retrieval and generation.

TypeScript SDKagent / RAG workflow
const proof = new ProofSource({
  apiKey: process.env.PROOFSOURCE_API_KEY,
  project: "acme-research-agent"
});

// 1. Find licensed sources within policy and budget
const sources = await proof.sources.search({
  query: userQuestion,
  maxPricePerCitation: "0.010",
  purpose: "commercial_research_answer"
});

// 2. Authorize the chosen source before model use
const auth = await proof.citations.authorize({
  sourceId: sources[0].id,
  agentId: "ResearchAgent-42"
});

// 3. Generate answer with licensed evidence
const answer = await llm.generate({
  question: userQuestion,
  context: auth.evidence
});

// 4. Verify citation and settle payment
const receipt = await proof.citations.verifyAndSettle({
  authorizationId: auth.id,
  answer,
  citationSpans: extractCitations(answer)
});

console.log(receipt.amount, receipt.status, receipt.hash);

Core API surface

Small set of endpoints, clear responsibility.

POST /sourcesPublish a source with licence terms, price, owner wallet, and verification metadata.
GET /sources/searchFind licensed sources matching a user query, budget, use case, and policy.
POST /citations/authorizeReserve permission to use a source before exposing it to the model.
POST /citations/verifyConfirm the source appears in the final answer and create proof of use.
POST /settlementsRelease USDC settlement after verified citation use.
GET /receipts/{id}Retrieve tamper-evident proof for publisher, buyer, or auditor.
